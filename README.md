# Customer Segmentation Analysis using Python

## 📌 Project Overview
This project focuses on Customer Segmentation Analysis using Python. The main objective is to analyze customer sales data and categorize customers into different segments based on their sales performance.

The project helps businesses identify:
- High Value Customers
- Medium Value Customers
- Low Value Customers

This analysis supports better business decision-making and customer targeting strategies.
